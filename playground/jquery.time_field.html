jquery.time_field.js

Turns an HTML input into a time-of-day picker.

Usage

  1. Include jQuery 1.3+ and jquery.time_field.js in your HTML
  2. Call $('input.time').time_field(); to transform all inputs with HTML class time into time-picker fields

As the user edits a time field, the field will trigger a time_field:time_changed event (on every keypress) as the value changes. The single argument to the event will be a 24-hour-clock String representation of the given time.

You may programmatically trigger a time_field:set_time event on the field with a new, 24-hour-clock String representation of a time, to set the field to use that new time.

Options:

on_invalid
String (or null, the default) which should be triggered when the user enters an invalid time string.

Inspiration: the time field is an autonomous component of jquery.calendar_date_picker.js.

Example: time_field:time_changed events

var $form = $('#example1');
$form.submit(function(e) { e.preventDefault(); });

var $src = $form.find('input');
var $sink = $form.find('tt');
var n = 0;

$src.bind('time_field:time_changed', function(e, time) {
	n += 1;
	$sink.text('(' + n + ') time_field:time_changed with arg "' + time + '"');
});

$src.time_field();
