Bodacity JavaScript Utilities

Each Bodacious JavaScript Utility is an independent entity. Browse to the utilities which interest you:

  • canonical_url.js transforms any URL (potentially relative) into an absolute URL, free of "../" path components.
  • channeler.js wraps jQuery's $.ajax() so that subsequent requests cancel earlier ones and event handlers can be registered independently.
  • jquery.excerpt.js trims a block-level element's text content to fit a set number of lines.
  • utf8.js presents a String as an Array of Numbers representing utf-8 bytes.
