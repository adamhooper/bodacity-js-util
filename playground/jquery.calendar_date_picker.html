jquery.calendar_date_picker.js

Picks a date in a calendar interface.

Usage

  1. Include jQuery 1.3+, jquery.calendar_date_picker.js, some CSS (this example includes some IE6-only CSS as well), and (optionally) jquery.time_field.js in your HTML
  2. Call $('div.calendar').calendar_date_picker(); to transform all divs with HTML class calendar into calendars. The divs will be emptied of any existing content in the process.

The calendar is a table of dates. When the user clicks one of those dates, a calendar_date_picker:date_selected event will fire with either a Date or null as argument.

You may also trigger a calendar_date_picker:select_date event on the div, passing a Date argument, to have it select that date (and fire the calendar_date_picker:date_selected signal).

Options:

previous_month_text
(default < Prev) Text for the "previous month" label.
next_month_text
(default Next >) Text for the "next month" label.
focus_date
(default today_date) Date on which the calendar should focus. In other words, the month displayed when the calendar first appears.
selected_date
(default null) Selected datetime, with minute or day precision.
today_date
(default now) Today's date (which gets a special HTML class).
month_texts
(default ['January', 'February', ...]) Text for month labels.
day_of_week_texts
(default ['S', 'M', ...]) Text for day-of-week labels.
time_text
(default undefined) String label for time-of-day text input. If undefined, do not allow time-of-day input.
allow_deselect
(default false) If true, clicking the selected date will change the selection to undefined.

Inspiration: is it just me, or are most date pickers frustrating to include in web pages? This date picker can be created programmatically and does not need to be associated with a text field (though it is easy to do so, using jquery.datetime_picker_field.js).

Examples

Example 1: date-only

var $div = $('#example1');
var $src = $div.find('div.calendar');
var $sink = $div.find('tt');
var n = 0;

$src.bind('calendar_date_picker:date_selected', function(e, datetime) {
	n += 1;
	$sink.text('(' + n + ') calendar_date_picker:date_selected with arg "' + datetime + '"');
});

$src.calendar_date_picker({
	allow_deselect: true,
	previous_month_text: '<',
	next_month_text: '>'
});

(calendar date picker will go here on page load)

Last event: (click in the calendar to fire events...)
